# fqbn

Arduino FQBN (fully qualified board name)

> ℹ️ [What's the FQBN string?](https://arduino.github.io/arduino-cli/dev/FAQ/#whats-the-fqbn-string)

## Install

```sh
npm install fqbn
```

## Usage

CommonJS:

```js
const { FQBN, valid } = require('fqbn');
```

TypeScript:

```ts
import { FQBN, valid } from 'fqbn';
```
